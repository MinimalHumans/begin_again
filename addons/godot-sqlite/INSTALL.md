# godot-sqlite Plugin

This project requires the godot-sqlite (GDSQLite) plugin for Godot 4.5+.

## Installation

1. Download from: https://github.com/2shady4u/godot-sqlite/releases
2. Extract the plugin into this `addons/godot-sqlite/` directory
3. Enable the plugin in Project > Project Settings > Plugins
4. The plugin provides the `SQLite` class used by DatabaseManager

## Required Files

After installation, this directory should contain:
- `plugin.cfg`
- `bin/` (platform-specific SQLite binaries)
- `gdsqlite.gd` or equivalent GDScript wrapper
